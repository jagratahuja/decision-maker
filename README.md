# Decision Helper 🧠

Decision Helper is a lightweight web tool designed to help users make decisions more logically.

Instead of choosing randomly between options, the tool evaluates choices using multiple criteria such as importance, urgency, long-term value, and enjoyment. It then calculates a score for each option and recommends the strongest choice.

This turns everyday decision-making into a structured process.

Live Website:
https://decision-jagratahuja.vercel.app/

---

## ✨ Features

- Add multiple decision options
- Evaluate options using weighted criteria
- Score calculation system
- Recommended option highlighting
- Clean minimal interface
- Responsive layout

---

## 🧠 Example Use Cases

- Should I study or take a break?
- Which project should I build next?
- What should I work on today?
- Which task should I prioritize?

---

## ⚙️ How It Works

1. Enter multiple decision options.
2. Rate each option using the evaluation criteria.
3. The system calculates a weighted score.
4. The option with the highest score is recommended.
